# 🏖️ SmartBeach API

Ένα ολοκληρωμένο, ασφαλές RESTful API (Backend) για τη διαχείριση κρατήσεων ξαπλωστρών σε Beach Bars. Το σύστημα υποστηρίζει πολλαπλούς ρόλους χρηστών (Πελάτες & Διαχειριστές), δυναμική δημιουργία πλεγμάτων (grids) παραλίας και οικονομικά στατιστικά.

## 🚀 Τεχνολογίες που χρησιμοποιήθηκαν
* **Γλώσσα:** Python 3
* **Framework:** FastAPI
* **Βάση Δεδομένων:** MySQL (mysql-connector-python)
* **Ασφάλεια:** JWT (JSON Web Tokens) για ταυτοποίηση, Bcrypt για κρυπτογράφηση κωδικών.

## ✨ Βασικά Χαρακτηριστικά (Features)

### 🔐 Σύστημα Ασφαλείας & Ταυτοποίησης (Auth)
* **Bcrypt Hashing:** Οι κωδικοί των χρηστών κρυπτογραφούνται πριν αποθηκευτούν στη βάση.
* **JWT Authentication:** Η είσοδος (login) παράγει ένα ψηφιακό κλειδί (token) που λήγει σε 60 λεπτά, το οποίο απαιτείται για τις κρατήσεις.
* **Role-Based Access Control (RBAC):** Διαχωρισμός δικαιωμάτων μεταξύ `user` και `admin`. Μόνο οι διαχειριστές έχουν πρόσβαση στα στατιστικά και τη δημιουργία παραλιών.

### 🏖️ Διαχείριση Παραλίας & Διαθεσιμότητα
* Δυναμική δημιουργία πλέγματος (grid) ξαπλωστρών με διαχωρισμό σε `premium` και `normal` θέσεις, με διαφορετική τιμολόγηση.
* Υπολογισμός διαθεσιμότητας σε πραγματικό χρόνο για συγκεκριμένες ημερομηνίες (αποτροπή διπλής κράτησης της ίδιας ξαπλώστρας την ίδια μέρα).
* Γεωγραφικά δεδομένα (Latitude/Longitude) για κάθε παραλία.

### 📅 Σύστημα Κρατήσεων (Bookings)
* Έξυπνος έλεγχος ημερομηνιών: Οι κρατήσεις επιτρέπονται αυστηρά μόνο κατά τη θερινή σεζόν (Μάιος - Οκτώβριος).
* Check-in / Check-out: Δυνατότητα κράτησης και ακύρωσης από τον πελάτη (προστατευμένα endpoints μέσω JWT).

### 📊 Admin Dashboard
* Αυτόματος υπολογισμός ημερήσιων εσόδων (Revenue) ανά παραλία.
* Παρακολούθηση ιστορικού κρατήσεων ανά χρήστη.

---

## 🗄️ Δομή Βάσης Δεδομένων (Relational Database)
Η βάση (`smartbeach`) αποτελείται από 4 αλληλένδετους πίνακες με `ON DELETE CASCADE` για ακεραιότητα δεδομένων:
1. **`users`**: Στοιχεία πελατών/διαχειριστών, κρυπτογραφημένοι κωδικοί και ρόλοι.
2. **`beaches`**: Ονομασίες, περιγραφές και συντεταγμένες Beach Bars.
3. **`sunbeds`**: Τοποθεσία ξαπλώστρας στο grid (x, y), τύπος (premium/normal) και τιμή.
4. **`reserves`**: Το ιστορικό κρατήσεων με `UNIQUE KEY` στην ξαπλώστρα και την ημερομηνία.

## 📡 Βασικά API Endpoints

### 👤 Authentication & Users
* `POST /register`: Εγγραφή νέου χρήστη ή Admin.
* `POST /login`: Σύνδεση και έκδοση JWT Token.

### 🌊 Αναζήτηση & Πληροφορίες (Public)
* `GET /beaches`: Επιστρέφει όλες τις διαθέσιμες παραλίες.
* `GET /beaches/{beach_id}/sunbeds`: Λίστα με όλες τις ξαπλώστρες μιας παραλίας.
* `GET /beaches/{beach_id}/sunbeds/availability`: Διαθεσιμότητα ξαπλωστρών για συγκεκριμένη ημερομηνία.
* `GET /beaches/{beach_id}/sunbeds/count`: Στατιστικά χωρητικότητας παραλίας (σύνολο, premium, normal).

### 🎫 Κρατήσεις (Απαιτείται Token Πελάτη)
* `POST /book`: Δημιουργία νέας κράτησης.
* `DELETE /book`: Ακύρωση υπάρχουσας κράτησης.

### 👑 Διαχείριση (Απαιτείται Δικαίωμα Admin)
* `POST /admin/beaches/{beach_id}/sunbeds/grid`: Δημιουργία χάρτη ξαπλωστρών.
* `GET /admin/revenue`: Υπολογισμός συνολικών εσόδων από τις κρατήσεις.
* `GET /admin/users/{user_id}/bookings`: Προβολή ιστορικού πελάτη.
