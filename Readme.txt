# 🏖️ SmartBeach - Full-Stack Booking System

Ένα ολοκληρωμένο, ασφαλές σύστημα κρατήσεων (Full-Stack) για Beach Bars. Η εφαρμογή επιτρέπει σε πελάτες να κάνουν κράτηση ξαπλώστρας με ακριβείς ώρες άφιξης/αναχώρησης, ενώ παρέχει στους διαχειριστές (Admins) ένα ισχυρό περιβάλλον (Dashboard) για τη δυναμική δημιουργία πλεγμάτων παραλίας, τη διαχείριση χρηστών και την παρακολούθηση εσόδων.

## 🚀 Τεχνολογίες (Tech Stack)
* **Frontend:** React Native, Expo, Expo Router (TypeScript)
* **Backend:** Python 3, FastAPI
* **Βάση Δεδομένων:** MySQL (με `mysql-connector-python`)
* **Ασφάλεια:** JWT (JSON Web Tokens) για Authentication, Bcrypt για Hashing κωδικών.

---

## ✨ Βασικά Χαρακτηριστικά (Features)

### 🧑‍💻 Για τους Πελάτες (Users)
* **Ασφαλής Εγγραφή & Σύνδεση:** Κρυπτογραφημένοι κωδικοί και προστασία API διαδρομών (endpoints) με JWT.
* **Διαθεσιμότητα σε Πραγματικό Χρόνο:** Προβολή διαθέσιμων ή κλεισμένων ξαπλωστρών για οποιαδήποτε ημερομηνία.
* **Ακριβείς Κρατήσεις:** Επιλογή ημερομηνίας, **ώρας άφιξης** και **ώρας αναχώρησης**.
* **Ιστορικό & Διαχείριση:** Προβολή προσωπικών κρατήσεων, αλλαγή ωραρίου κράτησης και δικαίωμα ασφαλούς ακύρωσης.

### 👑 Για τους Διαχειριστές (Admins)
* **Δυναμική Γεννήτρια Πλέγματος (Grid Generator):** Αυτόματη δημιουργία χάρτη ξαπλωστρών (Rows/Cols) με διαφορετική τιμολόγηση για Premium/VIP και Κανονικές θέσεις.
* **Διαχείριση Παραλιών:** Full CRUD (Δημιουργία, Επεξεργασία, Προβολή) πληροφοριών και γεωγραφικών συντεταγμένων.
* **Διαχείριση Χρηστών:** Προβολή όλων των εγγεγραμμένων χρηστών και δυνατότητα αναβάθμισης απλών χρηστών σε Admins.
* **Έλεγχος Κρατήσεων:** Προβολή του ιστορικού κρατήσεων όλης της πλατφόρμας, με δικαίωμα ακύρωσης ή επεξεργασίας ωρών.
* **Οικονομικά Στατιστικά:** Αυτόματος υπολογισμός και προβολή συνολικών εσόδων (Revenue) ανά παραλία.

## 🛠️ Οδηγίες Εγκατάστασης (Local Setup)

Για να τρέξετε το project τοπικά στον υπολογιστή σας, ακολουθήστε τα παρακάτω βήματα:

### Προαπαιτούμενα
* [Node.js](https://nodejs.org/) & npm
* [Python 3.8+](https://www.python.org/)
* [MySQL Server](https://dev.mysql.com/downloads/installer/) (π.χ. μέσω XAMPP ή MySQL Workbench)
* Εφαρμογή **Expo Go** στο κινητό σας (iOS / Android).

---

### Βήμα 1: Στήσιμο Βάσης Δεδομένων (Database)
1. Ανοίξτε το MySQL περιβάλλον σας (π.χ. phpMyAdmin).
2. Δημιουργήστε μια νέα βάση δεδομένων με το όνομα `smartbeach`.
3. Εκτελέστε τον κώδικα του αρχείου `Beaches.sql` για να δημιουργηθούν οι πίνακες και τα δεδομένα.
4. **Σημαντικό:** Βεβαιωθείτε ότι έχετε τρέξει τις παρακάτω εντολές για να προστεθούν οι στήλες της ώρας στον πίνακα των κρατήσεων:
   ```sql

   ALTER TABLE reserves ADD COLUMN leave_time VARCHAR(10) DEFAULT '18:00';
